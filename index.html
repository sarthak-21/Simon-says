For Instructions

How to play the Game?


Get ready with the mouse in your hand and click on the blinked button to follow the Pattern. Test how long you can memorize the pattern.


Rules of the Game

Game: Simon Says

For 1 or More Players.


Object of the game

Correctly repeat a longer and longer sequence of signals



1-Player Game

  1. Slide the ON/OFF switch to ON, the Game Selector switch to "1" and the Skill Level switch to 1,2, 3 or 4.
  2. Press any key to start playing the game. Simon will give the first signal.
  3. Repeat the signal by pressing the same color lens and the Skill Level switch to 1,2, 3 or 4.
  4. Simon will duplicate the first signal and add one.
  5. Repeat these two signals by pressing the same color lenses, in order.
  6. Continue playing as long as you can repeat each sequence of signals correctly.
  7. If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound.
  8. This means you've lost, and the sequence of signals ends. You can start over by pressing any key.

End of the Game

Repeat the number of sequences for Skill Levels 1, 2, 3 or 4 and Simon will salute you with six short signals from the last lens you pressed.


2 or More Player Game

Player may team up or compete with each other

  1. Decide who goes first.
  2. Play proceeds as described above except players take turns repeating Simon's signals.
